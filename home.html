Web views tracking

This is a small side project that I made that you can use to count your website views.
Thanks for using this

How to use

To use this, put the HTML code the app give you to the < head >.
Please do not rely on this app because I can clean the file anytime without warning. (But I'll not clean really often)
Create a view tracking code

Command Line Interface

To use with CLI, use http post method with the folowing post data to https://Web-views-tracking.lukenk.repl.co/cli:

"respond" : [your_option]

There are 2 option:
[link] Respond the entire link (include the app link) to put in HTML tag
[code] Respond only 10 characters for the tracking code
[html] Respond the HTML tag
[count] Respon the link to the counter

Security and reliable

This is an open source project, so everyone can see the data you stored.
Please do not rely on hosting option because I can clean the file anytime without warning.
